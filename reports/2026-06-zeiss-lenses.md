# 一片蔡司镜片：1889 年就把分配规则写进章程的公司，怎么对待 2026 年的广州工人

**调查对象**：蔡司眼镜镜片（卡尔蔡司光学，广州黄埔/中新知识城工厂生产，蔡司全球最大制造基地）
**调查日期**：2026-06-11
**案例定位**：第五案例——**基金会企业**（Stiftungsunternehmen），一种既非上市、也非创始人独资的治理结构；同时是「跨法域对照」的镜像案例（福耀：中国公司开美国厂；蔡司：德国公司开中国厂）
**调查方式**：纯公开信息聚合（集团年报 IFRS、基金会章程史料、招聘信息、工劳网判决书库）

置信度图例：🟢 直接证据 ｜ 🟡 间接证据 ｜ 🔵 推断估算 ｜ ⬜ 数据缺口

> **本案例的特殊性**：蔡司未上市，但因发行债券等原因按 IFRS 编制并公开经审计的集团年报——**它证明「非上市」和「可审计」不是互斥的**。这直接修正了我们此前从胖东来案例得出的「非上市 → 财务数字上限 🟡」的经验：那是中国民企的现状，不是结构必然。

## 一、链路还原

```
光学树脂/玻璃原料（⬜）
    ▼
卡尔蔡司光学（中国）有限公司 —— 广州中新知识城基地，员工 2000+
    │  车房镜片/成品镜片生产；两班倒、无尘车间
    │  一线底薪 2,600~2,700 元/月（≈广州最低工资线上方 4~8%）
    ▼
蔡司视力保健事业部（全球） ←── Carl Zeiss AG（总部德国 Oberkochen，员工 22,857 在德国）
    │                              ▲
    │                              │ 唯一股东：卡尔蔡司基金会（1889 年设立）
    │                              │ 章程禁止出售股份；分红用于资助科学
    ▼
全国眼镜店/视光中心（蔡司镜片零售常见数百~数千元/副）
    ▼
消费者
```

| 链路证据 | 置信度 | 来源 |
|---|---|---|
| 广州中新知识城厂区为蔡司**全球最大制造基地**，视力保健+医疗技术，员工 2000 余人 | 🟢 | [校招宣讲页](https://m.haitou.cc/xjh-2042250) |
| 卡尔蔡司基金会是 Carl Zeiss AG 唯一股东，章程禁止出售股份 | 🟢 | [基金会官方说明](https://www.schott.com/zh-cn/about-us/company/carl-zeiss-foundation) |
| 镜片零售加价链（眼镜店环节常见 3~10 倍加价） | 🔵 | 行业常识，本次未展开 |

**结构性事实**：这是五个案例中唯一一家**所有权被锁死**的公司——基金会不能卖、不能上市、不能被收购。1896 年阿贝章程同时写死了对员工的义务。但章程的效力范围，和胖东来的善意一样，有边界——见第三节。

## 二、价值分配（财报三件套，IFRS 经审计，全部 🟢）

### 2024/25 财年（截至 2025-09）

| 指标 | 数值 |
|---|---|
| 营收 | 118.96 亿欧元（+9%；德国本土仅占 6%） |
| EBIT | 15.52 亿欧（13%） |
| 合并净利润 | 9.52 亿欧 |
| 员工（期末） | 46,622 人（德国 22,857 / 海外 23,765） |
| **人员成本总额** | **43.49 亿欧**（工资 35.72 亿 + 社保 6.02 亿 + 养老金 1.74 亿） |
| 研发投入 | 17.31 亿欧（营收的 15%） |
| 执行董事会薪酬合计 | 1,973 万欧（含养老金服务成本；监事会另 110 万欧） |
| 利润去向 | 不分给私人股东——分红进入基金会，资助科学研究 |

来源：[蔡司 2024/25 年报](https://www.zeiss.com/annual-report/key-figures.html)、[财年新闻稿](https://www.zeiss.com/corporate/en/about-zeiss/present/newsroom/press-releases/2025/end-of-year.html)

### 核心比值 🟢/🔵

| 指标 | 蔡司 | 对照 |
|---|---|---|
| **劳资分配比**（净利润 ÷ 人员成本） | **≈ 0.22** | 蜜雪 2.86 ｜ 福耀 1.34 ｜ 胖东来 ≈0.03 |
| 人均薪酬（年均 45,702 FTE） | ≈ 9.5 万欧/年 | 福耀 ≈ 16.3 万元人民币 |
| 人均为"股东"创利 | ≈ 2.1 万欧/年 | 且这个"股东"是公益基金会 |
| 董事会人均 ÷ 集团人均薪酬 | ≈ **35 倍**（董事会若干人合计 1,973 万欧） | 福耀曹德旺 ≈ 97 倍（对一线） |

**0.22 的含义**：每产生 1 欧元归基金会的利润，员工已先拿走约 4.6 欧元。结构上接近胖东来端，且数字经过审计——**这是目前唯一一个「分配友好 + 可证伪」同时成立的案例**。

但有一个阿贝章程管不住的数字：**历史上阿贝规定管理层薪酬不得超过工人平均工资的约 10 倍**（🟡，史料记载）；今天董事会人均约为集团人均的 35 倍。基金会结构挡住了股东拿走利润，没挡住一百多年里高管薪酬相对工资的通胀——尽管 35 倍在跨国公司里仍属克制（美式大公司 CEO 对一线常见 200~400 倍）。

## 三、劳动证据墙

### 3.1 墙内：德国（章程与法律的覆盖区）

| 证据 | 置信度 | 来源 |
|---|---|---|
| **1900 年试行、1901 年正式实行八小时工作制**——比德国法定（1918）早 17 年；阿贝的口号"8 小时工作、8 小时睡眠、8 小时做人" | 🟢 | [阿贝文集（古腾堡计划）](https://www.gutenberg.org/files/19755/19755-h/19755-h.htm)、史料 |
| 1896 阿贝章程：保底工资 + 利润分享、带薪休假、病假发 3/4 工资、5 年工龄起养老金、**工龄 3 年以上因公司原因解雇至少补偿半年工资**、工人代表会与厂方对话机制 | 🟢 | 同上 + [企业史料](https://www.schott.com/zh-cn/about-us/company/history/corporate-history) |
| 现行：德国员工 22,857 人，适用德国《共同决定法》（监事会含劳方代表）及金属行业集体谈判体系（IG Metall，巴符州协议周工时 35 小时） | 🟢/🟡 | 德国法定框架 + 行业惯例 |

### 3.2 墙外：广州（2026 年在招岗位，🟢 招聘方自述）

| 证据 | 置信度 | 来源 |
|---|---|---|
| 一线操作工**底薪 2,600~2,700 元/月**——广州最低工资约 2,500 元（🔵），底薪仅高出 4~8% | 🟢 | [校招页](https://m.haitou.cc/xjh-2042250)、[鱼泡直聘](https://www.yupao.com/zhaogong/387205934.html) |
| **官方校招工资表自列：月加班 114 小时**（工作日 66h + 周末 48h），加班费 1.5/2/3 倍合规计算，月薪合计约 6,573 元——法定加班上限为 36 小时/月 | 🟢 | 校招页工资表原文 |
| 两班倒、无尘服、**18~40 岁年龄门槛**、"二次入职不接收" | 🟢 | 招聘启事 |
| 福利确实优于本土工厂：六险一金 + 商业险、年终双薪、免费食宿、带薪病假/年假、配镜福利、免费体检 | 🟢 | 同上 |
| 经第三方人力公司招聘（劳务中介环节存在） | 🟡 | 招聘渠道特征 |
| **工劳网检索"卡尔蔡司"命中 19 条**，其中约 12 条为蔡司中国实体真实案件：张查理案两轮（2016~2017）、李冬案两轮（2016~2017）、朱杰峰案两审（2021）、廖青华诉太阳镜片公司（2019）等，被告集中于广州光学/光学科技两家工厂 | 🟢 | [工劳网检索结果](https://search.laborinfocn2.com/lawcases?q=卡尔蔡司) |

### 3.3 必须并排放的两组数字

> **同一家公司**：德国工人协议周工时约 35 小时；广州工厂的官方校招工资表，把「月加班 114 小时」（≈法定上限的 3.2 倍）作为收入构成印出来——不加班，底薪 2,600 元在广州无法生活。
>
> **同一部章程**：1901 年让蔡司成为全球第一批八小时工作制企业；2026 年，八小时在广州厂区是「底薪 2,600」的同义词，凑齐 6,500 元月薪靠的是把工时拉回阿贝改革前的水平（阿贝接手前的蔡司：每天 13 小时——广州厂 8+3.8 小时加班 ≈ 11.8 小时）。

## 四、矛盾点对照

| 宣传/叙事 | 公开证据 |
|---|---|
| 基金会企业典范、"不受股东利益影响"、阿贝章程是"德国经济与社会史上最重要的文件之一" | 章程的劳动条款效力止于德国法域。广州工人的待遇由中国《劳动法》下限 + 加班结构决定，而非阿贝标准（🟢） |
| 八小时工作制发明者之一（1901） | 广州官方校招工资表自列月加班 114 小时，超法定上限 3.2 倍（🟢）。合规付加班费 ≠ 合规工时总量 |
| "蔡司全球最大制造基地"落地广州，宣传技术与品质 | 选择广州的成本逻辑中，底薪贴最低工资线 + 弹性加班结构是要素之一；德国本土营收占比仅 6%，但德国员工占 49%——**高薪岗位留在德国，工时密集环节放在墙外**（🟢/🔵） |
| 利润不归私人股东、资助科学 | 真，且经审计。但基金会模式回答的是"利润归谁"，不回答"利润形成过程中工时怎么分布"——0.22 的分配比与 114 小时的加班可以同时成立（🟢） |
| 高管薪酬克制（阿贝曾设 ~10 倍上限） | 现董事会人均约为集团人均薪酬 35 倍——远好于行业，但章程原意已松动（🟡/🔵） |

## 五、调查结论（证据指向，但请读者自行判断）

1. **基金会企业是第三种治理结构的实证**：不上市（无股东压力）、也不依赖在世创始人的个人意志（阿贝 1905 年去世，章程运转至今 130 年）。劳资分配比 0.22 且经审计——它证明「分配友好」可以被**制度化并跑过一个多世纪**，这是胖东来模式最缺的那块拼图：不依赖于东来活着。
2. **但章程出不了法域，和善意出不了墙是同一件事**：阿贝章程管得住 Oberkochen，管不住广州。广州工人拿的是「中国制造业合规线 + 外企福利溢价」：六险一金、年终双薪是真的，底薪贴最低工资线、月加班 114 小时也是真的。**蔡司没有把 1896 年的章程精神出口到它最大的制造基地，它出口的是德国质检标准。**
3. **与福耀互为镜像**：福耀把中国工时文化带去美国，被 NLRB 和工会顶回来；蔡司把生产放到中国，自动继承了这里更弱的工时约束。两个方向共同证明：**跨国公司的劳动条件跟着当地法域走，不跟着母公司价值观走**——无论母公司是曹德旺还是一个公益基金会。
4. **「非上市 = 不可审计」被证伪**：蔡司不上市但发布 IFRS 审计年报。胖东来的不透明是选择（或中国民企生态），不是非上市的宿命。这条要回写进方法论。

## 附：方法论复盘

**跑通了什么**：
- 财报三件套对德国基金会企业完全适用（年报附注列示人员成本、董事会薪酬、分地区员工数）
- 工劳网对外企中国实体同样有效（19 条命中，约 12 条直接相关）
- **招聘启事的「工资构成表」是高价值证据**：广州校招页自己列出月加班 114 小时——比任何爆料都硬，因为是雇主自述（🟢）
- 新增对照范式确认：跨法域对照可以双向使用（中国公司出海 / 外国公司入华）

**断裂点**：
- 蔡司中国实体单独的财务与薪酬总额不可得（集团合并报表不分国别列人员成本）
- 现行基金会章程中高管薪酬条款的确切内容未核实（10 倍上限是否正式废除、何时废除 ⬜）
- 树脂原料、眼镜店零售环节未展开
- 德国工厂一线实际工资未取得一手数据（依赖 IG Metall 协议框架推断 🟡）
