# 你的 Model Y 上那套福耀玻璃，是谁磨出来的

**调查对象**：国产特斯拉 Model Y 所用福耀汽车玻璃（整车 26.35 万元起，玻璃出厂价估算 1500~2300 元/套）

## 一、链路还原：从砂矿到你的天幕

```
硅砂矿（福耀自有：海南文昌、湖南溆浦、辽宁本溪三大硅砂厂）
    │  硅砂约占浮法玻璃成本 12%；纯碱（占19%）、天然气（占32%）市场采购
    ▼
浮法玻璃基地（福清、重庆、通辽、本溪 4 处，自供率 80%+）
    │  浮法玻璃约占汽车玻璃成本的 35~40%
    ▼
汽车玻璃深加工工厂（福清等 16 省市基地：预处理→印刷→烘弯→合片→包边）
    │  普工两班倒/三班倒、站立作业、底薪+计件、月休 2~4 天
    ▼
特斯拉上海超级工厂（总装）
    ▼
你的 Model Y（26.35 万元起）
```

**结构性事实**：福耀做到了**极致的垂直整合**——硅砂自采、浮法自产、设备模具自制、PVB 自产，
这是它毛利率（37.27%）碾压同行的根源。🟢 换句话说：这条链路上几乎没有"别的公司"，
从矿到玻璃，每一道工序的劳动都发生在福耀体系之内。**链条越短，利润越不需要分给别人。**

## 五、调查结论（证据指向，但请读者自行判断）

1. **垂直整合 = 利润内化，也 = 劳动内化**。福耀从矿山到成品不假外人之手，37% 的毛利率是教科书级的管理成就；同一枚硬币的反面是：链条上所有的"血汗"也都集中在同一个雇主名下，无处外推、也无人分摊议价权。
2. **93.12 亿 vs 69.24 亿**：股东一年拿走的比全体员工工资总和多 24 亿。这不是违法，这正是合法的、被资本市场奖励的分配结构（ROE 25.56% 是优等生的成绩单）——本报告想呈现的正是"合法"与"公平"之间的缝隙。
3. **同一个老板，两种待遇结构**：美国工人的安全问题值 10 万美元罚款和一部奥斯卡纪录片；福清工人的"月休 2~4 天"只是招聘启事里一行平静的小字。决定差异的不是雇主良心，而是工人所处制度环境给的筹码。
4. **慈善与分配是两个问题**：捐 300 亿建大学、修庙宇、赈灾，都是真的善行；但慈善是利润分配完成**之后**的自愿行为，它不回答"利润分配本身是否公平"这个问题，反而常被用来代替回答。

## 附：本次调查的方法论复盘（给血汗 agent 产品化的输入）

**比上一案例（蜜雪冰城）跑得更好的**：
- 年报附注里的"应付职工薪酬"科目挖出了工资总额 69.24 亿，与净利润形成本报告最有力的对照——**财报附注是被低估的证据富矿**，应纳入标准核查清单
- 找到了"同一雇主跨法域对照"这一新调查范式（中国厂 vs 美国厂），杀伤力极强
- 纪录片/OSHA 处罚等海外公开记录可以反哺对国内环节的理解

**断裂点**：
- 产线工人与白领的工资无法从"工资总额"中拆分，人均数据被高管薪酬拉高
- 单套玻璃出厂价只能用行业通用 ASP 估算，福耀对特斯拉的实际供货价是商业机密
- 特斯拉上海工厂环节（总装工人）本次未调查，链路在玻璃出厂后即移交
- 高管真实所得的大头是持股分红，不在"薪酬"科目里，需要另行核算股息流向

**修订记录（2026-06-11 第二版）**——应用两项方法论改进：
1. **财报附注三件套设为必查项**：补入董事+高管报酬合计 3,699.77 万元（曹德旺 776.19 万），与工资总额、员工人数构成完整的"薪酬阶梯"表——体系内从普工到曹德旺薪酬差约 97 倍
2. **裁判文书替代入口跑通**：天眼查/企查查司法风险页需登录、不可直接抓取；改用[工劳网劳动争议判决文书库](https://search.laborinfocn2.com/lawcases)（180 万份判决书，2008~2023）。检索"福耀玻璃"命中 71 份，其中 (2016)渝0110民初4421号判决书**把"12 小时两班倒"从员工爆料（🟡）升级为司法认定事实（🟢）**，并额外揭示了劳务派遣用工结构；另命中工伤系列案与多起直接劳动争议案。该库 2023 年后无数据，近年案件仍是盲区。
