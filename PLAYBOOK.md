# 血汗调查手册（Playbook v1.0）

**这份手册是什么**：一份可独立执行的供应链劳动调查方法论。任何人 + 任何能联网检索的 LLM（或纯人工），照着本手册操作，就能对一个产品产出与本仓库 `reports/` 同等结构和证据标准的调查报告。

**手册脱离本仓库也应成立**。所有规则基于五个已完成案例提炼：蜜雪冰城柠檬水、福耀玻璃（Model Y）、胖东来自由·爱白酒、永辉咏悦汇小酌、蔡司镜片。范例本身是手册的一部分——新手先读两份报告再开工，产出下限会高很多。

---

## 0. 立场纪律（先于一切技术）

1. **呈现证据 + 标注矛盾，不打分、不下结论。** 结论段的标准措辞是"证据指向，但请读者自行判断"。
2. **只用公开数据**：财报、判决书、行政处罚、招聘信息、媒体报道、企业自己的宣传。不做暗访、不收爆料、不做举报平台。
3. **诚实于断裂**：查不到就写"链路在此处断裂"，这本身是事实。禁止用推测填补。
4. **好公司同等力度地挖。** 对照组（如胖东来）的反面证据必须和血汗案例同样认真找，否则项目沦为"好公司榜单"。
5. **正反主张举证标准一致。** 不因公司口碑好就放宽对正面叙事的怀疑，不因口碑差就放宽对负面爆料的采信。

## 1. 置信度体系（每个数字必须挂一个）

**规范形式为文字标记**：【直接】【间接】【推断】【缺口】。
（emoji 🟢🟡🔵⬜ 仅作可选装饰——终端、纯文本、部分渲染环境会乱码或不显示，文字在任何环境都成立。本仓库早期报告用 emoji 形式，图例已在各报告头部定义。）

| 标记 | 判定标准 |
|---|---|
| 【直接】 | 审计财报/招股书原文、司法判决书、行政处罚决定、**企业官方自述**（含招聘启事的工资构成表、官网宣传——用于钉死它自己）、严肃媒体实地调查 |
| 【间接】 | 招聘网站参考薪资区间、个人公开爆料、媒体转引的无审计数字（含创始人自述财务）、行业惯例 |
| 【推断】 | 基于公开数据的计算，**必须写出假设条件**（如"按期末员工数折算，实际人均更低因年内裁员"） |
| 【缺口】 | 查过但查不到。必须显式标出，不许沉默跳过 |

**三条铁律**：
- 数字打架时，低置信度服从高置信度；同级打架，并列呈现并指出矛盾
- 【间接】不许在结论里被当成【直接】引用（最常见的偷工减料）
- 非上市公司的财务自述上限【间接】——但注意蔡司案例的修正：**非上市 ≠ 不可审计**（蔡司发 IFRS 审计年报）。先查有没有审计报表，再降级

## 2. 调查流程（标准七步）

```
① 定产品 → ② 还原链路 → ③ 跑必查项 → ④ 算价值分配
→ ⑤ 砌证据墙 → ⑥ 做矛盾对照 → ⑦ 回填跨案例指标
```

### ① 定产品
选**单一具体产品**（4 元柠檬水、一片挡风玻璃、一瓶 75 元的酒），不选"某公司"。产品是消费者的入口，公司是调查的对象。优先选用的人多、链路有代表性的。

### ② 还原链路
画出"原料 → 制造 → 品牌 → 渠道 → 消费者"每一环的**公司实体名**（精确到注册名，后面查判决书要用）。每一环标注：谁雇佣这一环的劳动者？这决定了第 5 步去哪挖。

### ③ 跑必查项（见第 3 节）

### ④ 算价值分配
把零售价拆到每一环；把公司年度净利润和全员薪酬放在一起；算跨案例指标（第 6 节）。

### ⑤ 砌证据墙
按"薪资、工时、社保、合同、司法记录"五个维度，每条证据一行：内容 + 置信度 + 原始链接。

### ⑥ 做矛盾对照
左列：企业宣传/ESG/创始人言论的原话。右列：本次调查找到的证据。**只放对照得起来的**，牵强的矛盾点会毁掉整份报告的可信度。

### ⑦ 回填跨案例指标
单篇报告讲故事，指标表沉淀数据资产。格式见第 6 节。

## 3. 必查项操作指引（中国法域版）

### 3.1 财报附注三件套（上市公司核心武器）

去交易所/巨潮资讯/东方财富找**年报全文 PDF**（业绩公告不够，港股业绩公告常不列董事酬金）。查三个科目：

| 科目 | 在年报哪里 | 能算出什么 |
|---|---|---|
| 应付职工薪酬（附注） | 财务报表附注，看"本期增加"列 = 全年薪酬计提总额 | 劳资分配比（净利润÷全员薪酬）、人均薪酬 |
| 员工人数 | "公司治理/员工情况"章节，母公司+子公司合计 | 人均创利 vs 人均薪酬；与体系内劳动者总数对比 → 报表外规模 |
| 董监高薪酬 | "董事、监事、高管情况"章节 | 薪酬阶梯倍数（最高薪 ÷ 一线年薪） |

**进阶科目（案例确立的）**：
- **辞退福利**（应付职工薪酬附注内单列）：裁员规模的财务指纹。永辉案例：5.95 亿辞退福利 vs 1,139 万调改店分红，52 倍——一个科目戳穿一个叙事
- 社保费/公积金分项：与员工数相除可验证缴纳水平
- 劳务外包/派遣费用（若披露）：报表外劳动的线索

**坑**：
- 期末员工数 ≠ 年均人数。大幅裁员年份用期末数算人均薪酬会虚高，须注明（标【推断】）
- 口径要统一：薪酬总额"含社保福利"和"仅工资奖金"是两个数，跨案例对比时标明
- 净利润为负时劳资分配比失效 → 切换"风险对称性"视角：股东亏的是回报，员工被裁承担的是失业，两种风险不对称（永辉案例）

### 3.2 司法判决检索

**入口**：[工劳网劳动争议判决文书库](https://search.laborinfocn2.com/lawcases)，查询格式 `?q=公司名`。约 180 万份判决书，覆盖 2008~2023。

**操作**：
- 检索品牌名 + 核心子公司注册名（"福耀玻璃"和各地"XX 福耀"是不同实体；蔡司要查"卡尔蔡司"）
- 判决书的价值是**把爆料升级为司法认定事实**：福耀的"12 小时两班倒"在判决书里是法院认定（【直接】），比任何员工爆料都硬
- 看**结构**而非数量：蜜雪命中 4 案但被告全是加盟店个体户、品牌方零案件——这本身就是"用工外包到报表外"的证据
- **人均密度**比绝对数有意义：永辉 942 份 vs 胖东来 77 份，按员工基数折算密度相近（≈0.8 份/百人）——绝对数对比会误导

**坑**：
- 2023 年后无数据（裁判文书网收缩，工劳网是民间补位）
- 命中含无关案件（如"卡尔蔡司"会命中提及蔡司培训的第三方案件），须逐条甄别
- 天眼查/企查查司法风险页需登录，不可用作 agent 入口
- 店员/零工维权多止步于仲裁，进不了法院——零案件 ≠ 零纠纷

### 3.3 招聘信息（薪资工时最稳定的信源）

智联/BOSS/鱼泡/公司校招页。**重点找"工资构成表"**：

> 蔡司案例的发现：广州厂官方校招页自列「底薪 2600 + 工作日加班 66h + 周末加班 48h = 月薪 6573」——雇主自述月加班 114 小时（法定上限 36h 的 3.2 倍）。**这是【直接】级证据，因为是雇主自己印的**，比任何爆料都硬。

要点：
- 底薪 vs 当地最低工资的距离（贴线 = 收入靠加班撑）
- 班制（两班倒/三班）、月休、年龄门槛、"二次入职不接收"等筛选条款
- 招聘方是公司本体还是第三方人力中介（派遣/外包线索）

### 3.4 行政处罚
信用中国、地方人社/应急管理部门公示。少数能直接定性的证据类型。覆盖稀疏，查到是赚到。

### 3.5 企业宣传 / ESG 报告
官网、ESG/社会责任报告、创始人公开言论。这是**矛盾点对照的原材料**：先把企业说了什么原文摘下来，再去对证据。
- 关键指标：**ESG 承诺覆盖率** = 承诺覆盖的人数 ÷ 体系内全部劳动者。蜜雪"保障员工社保"只覆盖 9,102 名集团雇员，30 万加盟店店员不在内 → 覆盖率 ≈3%

### 3.6 媒体报道与爆料
严肃媒体实地调查可到【直接】；自媒体/个人爆料为【间接】，须找第二信源交叉。

## 5. 对照范式库（让证据自己说话的四种排法）

| 范式 | 操作 | 来源案例 |
|---|---|---|
| **跨法域** | 同一雇主在不同法域的工时/工资/工会待遇并排 | 福耀（中企赴美被 NLRB 约束）、蔡司（德企入华继承弱约束）——双向均可用，共同结论：劳动条件跟法域走，不跟母公司价值观走 |
| **墙内墙外** | 同一产品，零售端员工 vs 代工厂工人待遇并排 | 胖东来（店员 9,886 元 vs 宝丰工人 3000~4500 元，透明止于供价） |
| **纵向自身** | 同一公司变革前后并排 | 永辉（调改前底薪 2500 vs 调改后 4500~6000；分红 1,139 万 vs 辞退福利 5.95 亿） |
| **章程 vs 现实** | 公司的历史承诺/章程条款 vs 当下执行 | 蔡司（1901 年八小时制发明者 vs 2026 广州月加班 114h；阿贝 10 倍薪酬上限 vs 现 35 倍） |

## 6. 跨案例指标回填（血汗指标体系）

每个案例完成后，向 `cross-metrics.md` 回填五类指标。复刻者可直接复用该文件的表结构作为 schema：

- **A 分配类**：劳资分配比（净利润÷全员薪酬）、人均创利 vs 人均薪酬、薪酬阶梯倍数、单品劳/资含量
- **B 边界类**：报表外劳动者规模（边界外/内比）、ESG 承诺覆盖率
- **C 强度类**：班制、月休、推算月工时、最低时薪证据
- **D 保障类**：社保、合同、司法记录（看结构与人均密度，不比绝对数）
- **E 修辞类**：公司捐赠占净利比、宣传-证据矛盾点数

**口径纪律**：薪酬总额优先取"含社保福利"口径；不一致时在表注标明。

**已知失效场景**：
- 亏损公司：A1 为负 → 标"指标失效"，转风险对称性叙述
- 非上市无审计：A 类大面积【缺口】/【间接】→ 启用替代锚点（产品成本公示、判决书侧面事实、招标公告）

## 7. 治理结构变量（与置信度分开记录）

公司的治理结构独立影响劳动结果，**不是数据质量标签**。每个案例记录：

| 结构 | 代表案例 | 对劳动的典型影响 |
|---|---|---|
| 上市 + 加盟制 | 蜜雪 | 劳动外包到报表外，ESG 承诺覆盖率极低 |
| 上市 + 垂直整合 | 福耀 | 劳动留在报表内，工时强度内化 |
| 上市 + 转型期 | 永辉 | 资本纪律与改善并行：涨薪给留下的人，账单给离开的人 |
| 非上市 + 创始人意志 | 胖东来 | 可重写分配规则（95% 分员工），但不可继承、不可证伪 |
| 非上市 + 基金会章程 | 蔡司 | 分配规则制度化且可继承（130 年），但效力止于法域 |

## 8. 常见断裂点与处理

| 断裂点 | 处理 |
|---|---|
| 原料种植/开采环节 | 五案例均未突破。如实标【缺口】，不硬写 |
| 代工厂工人待遇 | 招聘参考区间（【间接】）+ 判决书碰运气；写"透明止于供价" |
| 非上市公司财务 | 自述【间接】+ 替代锚点；头部挂警告 |
| 加盟店/个体户用工 | 工劳网查个体户被告案件；卧底报道（严肃媒体可到【直接】） |
| 2023 年后司法数据 | 工劳网盲区，标明；用行政处罚/媒体补 |
| 调查对象体量太小搜不到 | 升级到母公司/代工方/同园区同行业横切 |

## 9. 跨法域复刻：信源对照表

方法论骨架（四段式、置信度、五类指标、对照范式）**全部可移植**，换的是信源层：

| 中国 | 美国 | 德国/欧盟 |
|---|---|---|
| 年报附注三件套（巨潮/东财） | SEC 10-K + DEF 14A（高管薪酬）+ 人力资本披露 | Bundesanzeiger 年报、CSRD 报告 |
| 工劳网判决书 | NLRB 案件库、PACER、州法院 | 劳动法院（Arbeitsgericht）判例 |
| 招聘网站工资表 | Indeed/Glassdoor、H-1B/PERM 工资数据库 | Kununu、行业集体协议原文（公开） |
| 行政处罚（信用中国） | OSHA/DOL/EEOC 执法数据库 | 各州劳动监察、欧盟 ESG 执法 |
| ESG 报告 | 10-K Human Capital + ESG 报告 | CSRD（强制且范围更大） |

进入新法域的第一步：**重建本地版必查项清单**，并摸清当地的"工劳网等价物"在哪、覆盖到哪年。

## 10. 法律与伦理纪律（复刻者最容易偷工减料的地方）

1. **每个【直接】都要经得起被指控方核验**——置信度纪律不是学术洁癖，是法律护甲。影响力越大，越会被起诉方逐条检验
2. 不指控、不定性犯罪；只并排呈现"它说的"和"公开记录显示的"
3. 引用判决书用案号；引用财报用科目原文；引用招聘用截图存档（信源会消失）
4. 个人爆料须匿名化处理，即使原帖公开
5. **信源塌方预案**：工劳网本身就是裁判文书网收缩后的补位，没人保证它永续。每个核心信源记录至少一个替代入口

## 11. 复刻启动清单（拿到本仓库后）

```
□ 读 README（立场与原则）
□ 读本手册
□ 精读两份范例报告（推荐：一份上市公司如福耀，一份非上市如胖东来）
□ 选第一个产品（用的人多、链路有代表性、最好是上市公司——信源全，先建立手感）
□ 按第 2 节七步流程跑
□ 写报告（第 4 节模板），每个数字挂置信度
□ 回填 cross-metrics（第 6 节 schema）
□ 复盘：本案例发现了什么新信源/新范式/新坑 → 回写本手册
```

最后一条最重要：**手册是活的**。辞退福利科目、招聘工资构成表、"非上市≠不可审计"——全是案例跑出来后回写的。复刻者的案例也应如此反哺。
